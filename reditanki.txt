Title: Leave edit instructions for an LLM directly on your Anki cards — it batch-processes them later (open source, not an add-on)

Hi, while absolutely abusing Anki learning Spanish I kept hitting cards I wasn't happy with — a bloated note I didn't want to rewrite by hand, an example sentence that missed the point of the word.

So I built a workflow where I can "talk to" my cards from inside Anki: my note types have a `user_feedback` field, and when a card annoys me during review I just type an instruction into it ("shorten this", "use a rioplatense example instead") and keep reviewing. Later I run one command, and an LLM processes all the annotated cards in a batch — it backs up the deck first, shows me every proposed change as old → new, and nothing is applied without my approval.

The difference from pasting cards into ChatGPT: it knows my deck's conventions (note types, style rules, what my deck considers a good card), so edits come back consistent with the rest of the deck.

Being upfront: **this is not an Anki add-on.** It runs outside Anki via AnkiConnect and Claude Code (MCP), so there's real setup involved — terminal, API access. All open source.

I use it daily. Would love feedback — does this workflow make sense to anyone else, or is there an add-on that already does this that I missed?
